# 🌟 Daily Motivation App

A visually rich and interactive quote generator built with React.
Designed to deliver inspiration with a smooth, modern UI and delightful micro-interactions.

---

## 🚀 Features

* 🎯 **Random Quote Generator**
  Fetches motivational quotes from an API.

* 🔠 **Dynamic Font Resizing**
  Automatically adjusts text size to perfectly fit the quote container.

* ❤️ **Like & Save Quotes**
  Save your favorite quotes using local storage.

* 👎 **Dislike & Remove**
  Instantly remove quotes from your saved list.

* 📋 **Copy to Clipboard**
  Copy quotes with a single click (only once per quote).

* 📤 **Share Support**
  Native sharing (if supported by browser).

* 🧠 **Persistent Storage**
  Liked quotes are محفوظ (saved) using localStorage.

* 🎨 **Custom UI Design**

  * Glassmorphism card
  * Neon glow effects
  * LondonHistory font for aesthetic quotes
  * Interactive hover tooltips

## ⚙️ Tech Stack

* ⚛️ React (Vite)
* 🎨 CSS (Custom styling, no frameworks)
* 📦 Local Storage API
* 🌐 DummyJSON Quotes API

---

## 💡 Key Concepts Used

* React Hooks (`useState`, `useEffect`, `useRef`)
* Controlled UI state (no unnecessary syncing)
* Conditional rendering
* Dynamic DOM measurement (auto-resizing text)
* Component-based architecture
